## Component States

### Answer Submitted State (Waiting for AI)
```
[Spinner] Reviewing...
Textarea is disabled (grayed out)
Submit Answer button is hidden
AI Review section shows loading state
```

### Awaiting Difficulty Selection
```
AI Review section displays with full content
Difficulty buttons are enabled and clickable
"Easy" button is highlighted with ring
Submit & Next Card button is enabled
```

### Submitting Result State
```
[Spinner] Submitting...
Difficulty buttons are disabled
Submit & Next Card button shows loading state
User cannot interact
```

## Button Styles

### Difficulty Buttons (Selected State)
- Ring: 2px offset with primary color
- Border: 2px solid (color-specific)
- Padding: p-3 (12px)
- Rounded: lg (8px)
- Transition: all 200ms

### Primary Action Buttons
- Background: Blue-600
- Hover: Blue-700
- Disabled: Gray-400
- Width: Full (w-full)
- Height: 40px (h-10)

### Secondary Buttons
- Background: Transparent
- Border: 1px Gray-200
- Hover: Gray-100
- Width: Full (w-full)
